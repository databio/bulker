# Bulker

**Archived** This python version of bulker is being replaced by a much more powerful version written in rust. You can find out more at https://bulker.io.

Bulker builds multi-container computing environments that are both **modular** and **interactive**. 

For details, see the [bulker documentation](https://bulker.io). You can find and share manifests at [bulker hub](https://hub.bulker.io).
